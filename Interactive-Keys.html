Project information

Objective

The objective of this project was to try new frameworks Paper JS and Howler JS.

Tech Stack

JavaScript, HTML, CSS, Howler JS, Paper JS and jQuery.

Description

  • The project was designed using JavaScript, HTML, Howler JS, Paper JS,CSS, Bootstrap and jQuery.
  • When a person press a alphabet keys a sound is playes and a pattern is displayed.
  • Sound is played using Howler JS.
  • Pattern is displayed using Paper JS.
  • Position and color of the pattern are displayed randomly.
